For the simulation, I do not allow for a drop in the true bilirubin. That is, consider a biliogy where the true underlying disease process starts flat and then turns upward, and so does th “true” bilirubin that is driving that process. As our set of prototypes, fit a hyperbola to log(bili), for everyone with 4 or more values. The equation is \(y= a + b[(x-c) + \sqrt{(x-c)^2 + .5}]\); \(a\) is the starting “flat”, \(b\) the right hand slope and \(c\) the inflection point. We are essentially using a broken line with a horizontal left hand portion; the hyperbola formula has first and second derivatives so optimizer routines are happier. The .5 causes the curve to take about 1-2 years to fully turn the corner. For a subject with no flat portion, they were already rising at enrollment, the optimum has an inflection at -infinity; this completely removes the bend. But that solution also drives the intercept down to an odd value. The function is essentially flat at -5, so force that as a minimum allowed value for the inflection point, i.e., 5 years before enrollment. This constraint won’t change the RSS much at all, but makes the results look a bit more sane.

hfun <- function(par, x, y, ss=TRUE, positive=TRUE) {
    inflect <- pmax(par[3], -5)
    if (positive) par[2] <- pmax(0, par[2])
    yhat <- par[1] + par[2]*((x-inflect) + sqrt((x-inflect)^2 + .5))
    if (!ss) yhat
    else sum((y-yhat)^2)
}

To create a simulation data set we sample with replacement from the set of 220 starting age + bilirubin trajectory sets. This pseudo subject is sampled at intervals, with noise added to the observed bilirubin commesurate with the earlier fits. The true HR is based on enrollment age, true bilirubin, and chosen coefficients; a scale factor turns this into absolute risk based on actual (time dependent) age. This results in a random survival time + random censoring pair, which truncates the followup for that observation.

Why the negative coefficient for expected in efit — do death rates decrease? This is an example of immortal time bias, and a mistake I have made before. The issue is that early death guarrantees a small expected (a death on day 2 accumulates no time), thus a small expected ``predicts’’ early death. Using it as a fixed offset is fine, and shows that the overall death rate for PBC subjects is about 9x Minn expected.

The efit3 fit ignores the expected and is a valid way to proceed. A second idea is to use coxph along with the cumulative hazard. In the same way that \(F(X) ~ U(0,1)\) when \(X\) is a random draw from a distrbution with cumulative distribution function \(F\), a random draw from the cumulative hazard has an exponentia(1) distribution. So we could use approx() + random exp draw + the empriical cumhaz to get times. But the coefs of efit3 and pfit are very similar, and the coxph cumhaz is reasonably linear, so it is easier to use the efit3 formula directly. We are trying for a death rates in the same ballpark, not an exact match.

The simulatation will use continuous bilirubin. In order to facilitate different bilirubin coefficients, we use centered bilirubin, that way if we change the coef the overall death rate doesn’t move so much. Since the real survival will be based on time dependent bilirubin the hazard for any given prototype subject won’t be linear; we track it at monthly discrete times. A monthly step function will be good enough. All the simulations will be censored at 14 years, so 0-14 yrs post enrollment.

meanbili <- mean(log(pdata$bili))
xtime <- seq(0, 14, length= 1+ 14*12) 
realsurv <- function(beta, ecoef=coef(efit3)[1:2], debug=FALSE) {
    n <- nrow(base) # number of bilirubin patterns
    hazard <- matrix(0, length(xtime), n)
    for (i in 1:n) {
        lbili <- hfun(unlist(base[i,2:4]), x=xtime, ss=FALSE)
        lbili <- pmin(lbili, log(40))  # .1- 41 is the range in pbcseq
        # log of baseline haz at the start of each month
        bhaz <- ecoef[1] + ecoef[2]*(base$age[i]+ xtime)
        hazard[,i] <- exp(bhaz + (lbili-meanbili)*beta)
    }

    # we want the cumhaz to start at 0
    chaz <- apply(hazard, 2, function(x)
        c(0, cumsum(x[-length(x)]))) * (1/12)  # width of interval
    if (debug) browser()
    exp(-chaz)
}

surv1 <- realsurv(1.3)  #   1.3 = approx coef of TD cox fit using log(bili)

Simulation

Create simulated data sets where each observation is a random draw from the population of true (starting age, bilirubin over time) sets, for each generate a survival and censoring time, and a set of visit dates. Visit times are based on observed visit intervals. The protocol for the study specified a visit at 6 months and yearly thereafter, below I sample from the observed “time to first” and “time between others” intervals, but capping the interval lengths.

interval1 <- with(pbcseq, unlist(tapply(day, id, function(x) {
    if (length(x) <2) NULL else x[2]-x[1]})))
interval2 <- with(pbcseq, unlist(tapply(day, id, function(x) {
    if (length(x) <3) NULL else (diff(x))[-1]})))

# limit the max
interval1 <- unname(interval1[interval1 < 400])
interval2 <- unname(interval2[interval2 < 1000])

sim1 <- function(k, survmat, id=k, debug=FALSE) {
    # simulate a single subject from prototype k
    visits <- cumsum(c(0, sample(interval1, 1), 
                     sample(interval2, 15, replace=TRUE)))/365.25
    ctime <- runif(1, 4, 14)
    # approx will complain in survmat[,k] ends in a string of zeros
    nz <- sum(survmat[,k] >0)
    if (nz < length(xtime)) {
        j <- seq.int(1, nz+ 1L)             
        stime <- approx(survmat[j,k], xtime[j], runif(1))$y
    }
    else stime <- approx(c(survmat[,k],0), c(xtime,15),  runif(1))$y
    dtime <- min(ctime, stime)
    death <- 1*(stime <= ctime)
    visit2 <- visits[(dtime > visits)]
    if (any(is.na(visit2)) || is.na(dtime) || length(visit2)==0) browser()
                     
    lbili <- hfun(unlist(base[k, 2:4]), x=visit2, ss=FALSE)  # true bilirubin
    bili  <- exp(lbili) * runif(1, .9, 1.1)        # observed bilirubin
    bili <- pmin(bili, 41) # no outliers   
    nrow <- length(visit2)
    bili3 <- cut(bili, c(0, 1, 4, 100), c("normal", "1-4", ">4"))
    # 4 level state
    temp <- ifelse(bili3[-nrow]== bili3[-1], 0, as.numeric(bili3[-1]))
    bstate <- factor(c(temp, 4*death), 0:4, 
                     c("none", "normal", "1-4", ">4", "death"))
    if (debug) browser()
    #create a counting process data set for this subject
    data.frame(id=rep(id, nrow), age= base$age[k],
               year1 = visit2, year2=c(visit2[-1], dtime),
               bili = bili,
               bili3 = bili3,
               ibili3 = rep(bili3[1], nrow),
               death = c(rep(0, nrow-1), death),
               bstate= bstate)
}

To summarize: each simulated subject is randomly drawn from a (starting age, bilirubin progression) pattern, there are 220 unique patterns to draw from. For simplicity, each bilirubin pattern is treated as a step function with \(14*12 = 168\) steps over the maximum 14 year followup. The true survival for each of the patterns follows a proportional hazards model with \[ \lambda_i(t) = \exp(-6.09 + .044*age + 1.3*(log(bili)-.58)) \]

We saw further above that the marginal estimates, which are the mean survival curves for those with initial bilirubin in the \(\le 1\), 1–4 and \(>4\) groups, look very similar to the KM for those three groups in the pbcseq data. That serves to show that our simulation world has some broad similarity to the sequenial PBC data, but detailed mimicry is neither necessary nor something we strived for.

The curves above compare the KM and multi-state AJ curves to the true margin for a single random draw of 4000 subjects. It’s primary purpose is to check for any gross errors in our data generation process.

Simulation

For the actual simulation we would like to estimate quantiles (5, 25, 50, 75, 95) for the KM, multistate AJ, and multistate coxph approach. We will do this for a sample size of 312, matching the original study. To have reasonable precision in the outer percentiles we will do 1000 simulations, and retain results at the monthly time points.

} else {
    curve2 <- array(0, dim=c(nsim, length(xtime), 3, 3),
                dimnames=list(NULL, NULL, levels(base$bili3),
                              c("KM", "AJ", "Cox")))
    for (i in seq(along=simdata2)) {
        tdata <- simdata2[[i]]
        km <- survfit(Surv(year1, year2, death) ~ ibili3, id=id, tdata)
        temp <- matrix(1- summary(km, times=xtime, extend=TRUE)$surv, ncol=3)
        curve2[i,,,1] <- temp

        aj1<- survfit(Surv(year1, year2, bstate) ~ 1, tdata, id=id,
                      p0=c(1,0,0,0), istate=bili3)
        temp <- summary(aj1, times=xtime, extend=TRUE) 
        curve2[i,,1,2] <- temp$pstate[,4]

        aj2 <- survfit(Surv(year1, year2, bstate) ~1, tdata, id=id,
                       p0=c(0,1,0,0), istate=bili3)
        curve2[i,,2,2] <- summary(aj2, times=xtime, extend=TRUE)$pstate[,4]

        aj3 <- survfit(Surv(year1, year2, bstate) ~1, tdata, id=id,
                       p0=c(0,0,1,0), istate=bili3)
        curve2[i,,3,2] <- summary(aj3, times=xtime, extend=TRUE)$pstate[,4]

        coxfit <- coxph(list(Surv(year1, year2, bstate) ~ 1,
                         0:4 ~ age/common+ shared), tdata, id=id,
                        istate=bili3)

        dummy <- data.frame(age=base$age)
        csurv1 <- survfit(coxfit, newdata=dummy, p0=c(1,0,0,0))
        temp <- summary(csurv1, times=xtime, extend=TRUE)$pstate[,,4]
        curve2[i,,1,3] <- rowMeans(temp)

        csurv2 <- survfit(coxfit, newdata=dummy, p0=c(0,1,0,0))
        temp <- summary(csurv2, times=xtime, extend=TRUE)$pstate[,,4]
        curve2[i,,2,3] <- rowMeans(temp)

        csurv3 <- survfit(coxfit, newdata=dummy, p0=c(0,0,1,0))
        temp <- summary(csurv3, times=xtime, extend=TRUE)$pstate[,,4]
        curve2[i,,3,3] <- rowMeans(temp)
        cat(" ",i)
    }
    saveRDS(curve2, file="curve2.rds")
}
